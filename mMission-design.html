Design Principles

01
Navy-first All surfaces use deep navy blue. No light mode variant.
02
Touch-optimized Minimum tap target 56px. Large fonts for kiosk readability.
03
Clear action hierarchy Three-tier buttons: cancel / secondary / primary.
04
Gradient for active state Active tabs and primary buttons use blue gradient to indicate selection.
05
Monospace for data All financial values and technical data use monospace font.
06
Subtle borders Borders use low-opacity white, not solid colors.
07
White text only No grey text on dark navy. Muted text uses rgba(255,255,255,0.55).
